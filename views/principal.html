Sobre o projeto

O objetivo deste trabalho é aplicar as habilidades dos integrantes da equipe em Programação Orientada a Objetos (P.O.O.) em uma aplicação web, visando o desenvolvimento de uma landing page com páginas de login e cadastro. A solução foi pensada para incorporar validações robustas tanto no lado do servidor, garantindo a integridade e segurança dos dados, quanto no lado do cliente, com a implementação de mecanismos como o reCAPTCHA para prevenir abusos. O sistema armazena os dados registrados em um banco de dados MySQL, com a possibilidade de, no futuro, migrar para uma solução de nuvem, permitindo que as informações possam ser acessadas de qualquer lugar. A metodologia de Programação Orientada a Objetos foi essencial para estruturar o projeto de maneira eficiente, tornando o código mais modular, reutilizável e de fácil manutenção, alinhado às melhores práticas de desenvolvimento de software. Durante o processo de desenvolvimento, a equipe fez uso de ferramentas modernas como Git & GitHub para o controle de versões, Trello para o gerenciamento ágil das tarefas e Scrum para garantir a entrega do projeto dentro do cronograma. Agradecemos à professora Pamela, que nos proporcionou o conhecimento necessário para aplicar esses conceitos, permitindo o desenvolvimento de habilidades essenciais para o mercado de trabalho, como colaboração, comunicação, gestão de tempo e análise de projetos. Este trabalho foi uma valiosa oportunidade de aprender e crescer, aprofundando também nosso conhecimento em tecnologias e metodologias que são amplamente utilizadas no desenvolvimento de software profissional.
